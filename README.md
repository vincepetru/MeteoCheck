# MeteoCheck
indicando il nome di una città è possibile visualizzare il meteo corrente e la previsione meteo a 5 giorni, utilizzando le API pubbliche di Open‑Meteo. L’app recupera le coordinate tramite geocoding, ottiene i dati meteo correnti (temperatura, vento, giorno/notte) e mostra le temperature minime e massime giornaliere per i cinque giorni successivi.
